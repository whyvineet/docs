---
title: "Bookend AI integration"
description: "Integrate with the Bookend AI embedding model using LangChain Python."
---

Let's load the Bookend AI Embeddings class.

```python
from langchain_community.embeddings import BookendEmbeddings
```

```python
embeddings = BookendEmbeddings(
    domain="your_domain",
    api_token="your_api_token",
    model_id="your_embeddings_model_id",
)
```

```python
text = "This is a test document."
```

```python
query_result = embeddings.embed_query(text)
```

```python
doc_result = embeddings.embed_documents([text])
```
