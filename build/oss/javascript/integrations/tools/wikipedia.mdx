---
title: "Wikipedia integration"
description: "Integrate with the Wikipedia tool using LangChain JavaScript."
---

The `WikipediaQueryRun` tool connects your agents and chains to Wikipedia.

## Usage

```typescript
import { WikipediaQueryRun } from "@langchain/community/tools/wikipedia_query_run";

const tool = new WikipediaQueryRun({
  topKResults: 3,
  maxDocContentLength: 4000,
});

const res = await tool.invoke("LangChain");

console.log(res);
```
