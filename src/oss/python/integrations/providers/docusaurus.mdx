>[Docusaurus](https://docusaurus.io/) is a static-site generator which provides
> out-of-the-box documentation features.


## Installation and setup


<CodeGroup>
```bash pip
pip install -U beautifulsoup4 lxml
```

```bash uv
uv add beautifulsoup4 lxml
```
</CodeGroup>

## Document loader

See a [usage example](/oss/integrations/document_loaders/docusaurus).

```python
from langchain_community.document_loaders import DocusaurusLoader
```
