Oracle AI Database is built for AI workloads where you query data by **meaning** (semantics), not just keywords. It combines **semantic search over unstructured content** with **relational filtering over business data** in one systemâ€”so you can build AI retrieval workflows (like RAG) without introducing a separate vector database and fragmenting data across multiple platforms.

### Why it matters

- **One system for AI + business data:** vectors and operational data live together.
- **Less fragmentation:** fewer moving parts than a separate vector store.
- **Database-grade guarantees:** apply security, transactions, scale, and availability to the same AI workload.

## Prerequisites

Install `langchain-oracledb`. The `python-oracledb` driver will be installed automatically as a dependency.

<CodeGroup>
```bash pip
pip install -qU langchain-oracledb
```
```bash uv
uv add langchain-oracledb
```
</CodeGroup>
